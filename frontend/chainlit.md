# 🔍 Agentic Web Research Assistant

Ask me anything and I'll search the web in real time, summarise the results,
and remember our full conversation across the session.

**How it works:**
1. Your question is routed — web search or from memory?
2. If searching: Tavily fetches the top 5 sources
3. Results are summarised and cited
4. You get a clean, grounded answer

**Powered by:** LangGraph · Tavily Search · SQLite Memory · LangSmith
